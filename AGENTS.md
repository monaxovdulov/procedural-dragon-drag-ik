# AGENTS.md

Руководство для AI-агентов и разработчиков по работе с этим проектом.

## Среда и инструменты

- **Менеджер зависимостей и запуск:** работа ведётся через [uv](https://github.com/astral-sh/uv).
  - Установка зависимостей: `uv sync`
  - Запуск приложения: `uv run main.py`
  - Добавление пакета: `uv add <package>`
  - Python версия задаётся в `pyproject.toml` (>=3.12) и при необходимости в `.python-version`.

Не используйте `pip install` или `python -m venv` — зависимости и окружение управляются через uv.

## Короткие напутствия AI-агенту

- Начни с `TASKS.md` и `docs.md`, дальше делай задачи по порядку `task1.md` -> `task2.md` -> `task3.md`.
- Держи изменения локальными и маленькими: правь ровно то, что требуется, без рефакторинга "впрок".
- После каждого логического шага запускай и проверяй вручную: `uv run main.py` (TAB/drag UI, F1..F7 пресеты, D debug).
- Помни про `reset()`: изменения `N/L/leg_pairs` требуют `dragon.reset()`, остальные параметры можно менять на лету.
- Если одновременно есть ползунки и пресеты, реши источник правды (обычно: пресет обновляет значения слайдеров).
