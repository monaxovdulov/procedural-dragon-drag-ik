# Task 8: Больше Пресетов-«Скинов» Для Визуального Сравнения

## Контекст

Сейчас в проекте мало пресетов, поэтому сложно быстро сравнивать, какой “образ” дракона (толщина тела, рёбра, волна, пропорции лап, скорость шага) выглядит лучше.

Нужно добавить несколько пресетов-скинов и быстрый способ переключать их в рантайме (без перезапуска).

## Цель

- Добавить **набор пресетов** (минимум 6) с заметно разным “внешним видом/ощущением”.
- Сделать **быстрое переключение** клавишами `F1..F7`.
- Показать на экране **имя активного скина** (чтобы можно было сравнивать и обсуждать).

## Файлы

- `main.py`

## Требования

### 1) Пресеты должны быть “визуально разными”

Вариировать параметры, которые реально влияют на картинку:

- Толщина тела: `radius_head`, `radius_tail`.
- Рёбра/чешуя: `rib_every`, `rib_samples`, `rib_arc`, `rib_t_scale`, `rib_width`.
- Волна: `wave_amp`, `wave_freq`, `wave_speed`.
- Лапы: `leg_upper`, `leg_lower`, `leg_thickness`, `leg_hip_offset`.
- Походка: `leg_spread`, `leg_back`, `step_threshold`, `step_duration`, `step_lift`, `stride_front`, `stride_back`, `min_stance_time`, `move_vel_eps`.

Важно: часть параметров “структурные” и требуют `reset()` (см. `HARD_KEYS`).

### 2) Soft vs Hard

- **Soft skins (предпочтительно для сравнения):** менять только безопасные параметры, чтобы переключение было мгновенным, без `dragon.reset()`.
- **Hard skins (опционально):** можно оставить 1-2 пресета, которые меняют `N/L/leg_pairs` (для экстремальных вариантов), но они будут делать `reset()`.

### 3) Слайдеры и пресеты не должны конфликтовать

Если UI-ползунки включены, применяй пресет через `apply_preset(..., sliders)` так, чтобы:

- пресет синхронизировал значения слайдеров,
- `apply_sliders()` в следующем кадре не “откатывал” параметры обратно.

### 4) Хоткеи `F1..F7` + подпись активного скина

- `F1..F7` переключают разные пресеты.
- Текущий пресет хранится как строка, например `dragon.active_preset_name`.
- Рендерить подпись в UI (например, рядом с заголовком, или в левом верхнем углу).

## Набор рекомендуемых скинов (пример)

Ниже варианты с понятными характеристиками; можно корректировать значения после ручного просмотра:

1. `noodle` (тонкий, волнистый, частые шаги)
2. `imperial` (толстый, тяжёлый, редкие шаги)
3. `agile` (быстро реагирует, короткий шаг, более “нервный”)
4. `heavy` (очень вязкий и массивный)
5. `cartoon` (выразительный, высокая дуга шага, заметная волна)
6. `armored` (больше “рёбер”, более жёсткий силуэт)
7. `centipede` (hard: много лап, если хочется)

## Definition Of Done

- `F1..F7` реально меняют ощущения и внешний вид (видно без дебага).
- На экране видно имя активного пресета.
- Если в пресете нет “hard keys”, переключение не делает `reset()`.
- После применения пресета значения в слайдерах соответствуют пресету.

## Ручная проверка

1. `uv run main.py`
2. Поочередно нажать `F1..F7`.
3. Убедиться, что подпись пресета меняется.
4. Подвигать ползунки, затем снова применить пресет: UI и поведение синхронны.
